# Connex Updates

Public update feed for `Connex Desktop`.

This repository is intended to contain only published update artifacts, not source code.

## Download site

The root `index.html` is a small public download page. It reads `windows/latest.yml`
directly, so the main download button always points to the latest published Windows
installer.

## Windows feed

The Render Static Site should publish:

- `windows/latest.yml`
- `windows/Connex-Setup-<version>.exe`
- `windows/Connex-Setup-<version>.exe.blockmap`
